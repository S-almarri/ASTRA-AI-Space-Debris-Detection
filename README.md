# ASTRA – AI-Based Space Debris Detection System

ASTRA is a graduation project that explores the use of Artificial Intelligence, Computer Vision, and Embedded Systems for space debris detection and autonomous navigation concepts.

## Technologies

- Python
- OpenCV
- YOLO
- TensorFlow
- MobileNet
- Raspberry Pi
- Linux

## Features

- AI-based object detection
- Computer Vision integration
- Sensor integration
- Autonomous navigation concepts
- Real-time testing and validation
